The Ant Media Server iOS SDK enables ultra-low latency WebRTC streaming in native iOS applications. Built with Swift/Objective-C compatibility and optimized for iPhone and iPad.

## Installation

### CocoaPods

Add to your `Podfile`:

```ruby
pod 'AntMediaSDK', '~> 2.9.0'
```

Then run:

```bash
pod install
```

### Swift Package Manager

Add the package in Xcode:

1. Go to **File** > **Add Packages**
2. Enter the repository URL: `https://github.com/ant-media/WebRTC-iOS-SDK`
3. Select version and add to your project

### Manual Installation

1. Download the latest release from [GitHub](https://github.com/ant-media/WebRTC-iOS-SDK)
2. Add `AntMediaSDK.framework` to your project
3. Link the framework in your target settings

## Configuration

### Info.plist Permissions

Add required permissions to your `Info.plist`:

```xml
<key>NSCameraUsageDescription</key>
<string>Camera access is required for video streaming</string>

<key>NSMicrophoneUsageDescription</key>
<string>Microphone access is required for audio streaming</string>
```

### Background Modes

Enable background audio if needed:

```xml
<key>UIBackgroundModes</key>
<array>
    <string>audio</string>
</array>
```

## Basic Usage

### Initialize the SDK

<CodeGroup>

```swift Swift
import AntMediaSDK

class ViewController: UIViewController {
    var client: AntMediaClient!
    let serverUrl = "wss://your-server:5443/WebRTCAppEE/websocket"
    
    override func viewDidLoad() {
        super.viewDidLoad()
        
        // Initialize client
        client = AntMediaClient.init()
        client.delegate = self
        client.setOptions(url: serverUrl, streamId: "stream1", token: "")
        client.setWebSocketServerUrl(url: serverUrl)
    }
}

// Implement delegate
extension ViewController: AntMediaClientDelegate {
    func clientDidConnect(_ client: AntMediaClient) {
        print("Connected to server")
    }
    
    func clientDidDisconnect(_ client: AntMediaClient) {
        print("Disconnected from server")
    }
    
    func clientHasError(_ client: AntMediaClient, message: String) {
        print("Error: \(message)")
    }
}
```

```objectivec Objective-C
#import <AntMediaSDK/AntMediaSDK.h>

@interface ViewController () <AntMediaClientDelegate>
@property (nonatomic, strong) AntMediaClient *client;
@end

@implementation ViewController

- (void)viewDidLoad {
    [super viewDidLoad];
    
    // Initialize client
    self.client = [[AntMediaClient alloc] init];
    self.client.delegate = self;
    [self.client setOptionsWithUrl:@"wss://your-server:5443/WebRTCAppEE/websocket"
                          streamId:@"stream1"
                             token:@""];
}

#pragma mark - AntMediaClientDelegate

- (void)clientDidConnect:(AntMediaClient *)client {
    NSLog(@"Connected to server");
}

- (void)clientDidDisconnect:(AntMediaClient *)client {
    NSLog(@"Disconnected from server");
}

- (void)clientHasError:(AntMediaClient *)client message:(NSString *)message {
    NSLog(@"Error: %@", message);
}

@end
```

</CodeGroup>

### Publishing a Stream

Publish video from the device camera:

<CodeGroup>

```swift Swift
// Set local video view
client.setLocalView(view: localVideoView)

// Start publishing
let streamId = "myStream123"
client.publish(streamId: streamId)

// Publish with token
let token = "your-publish-token"
client.publish(streamId: streamId, token: token)

// Stop publishing
client.stop()
```

```objectivec Objective-C
// Set local video view
[self.client setLocalViewWithView:self.localVideoView];

// Start publishing
NSString *streamId = @"myStream123";
[self.client publishWithStreamId:streamId];

// Publish with token
NSString *token = @"your-publish-token";
[self.client publishWithStreamId:streamId token:token];

// Stop publishing
[self.client stop];
```

</CodeGroup>

## Video Rendering

Add video views to your interface:

<CodeGroup>

```swift Swift
import UIKit

class ViewController: UIViewController {
    let localVideoView: UIView = {
        let view = UIView()
        view.backgroundColor = .black
        return view
    }()
    
    let remoteVideoView: UIView = {
        let view = UIView()
        view.backgroundColor = .black
        return view
    }()
    
    override func viewDidLoad() {
        super.viewDidLoad()
        
        // Add video views to your layout
        view.addSubview(localVideoView)
        view.addSubview(remoteVideoView)
        
        // Set constraints...
        
        // Set views in client
        client.setLocalView(view: localVideoView)
        client.setRemoteView(view: remoteVideoView)
    }
}
```

```objectivec Objective-C
@interface ViewController ()
@property (nonatomic, strong) UIView *localVideoView;
@property (nonatomic, strong) UIView *remoteVideoView;
@end

@implementation ViewController

- (void)viewDidLoad {
    [super viewDidLoad];
    
    // Create video views
    self.localVideoView = [[UIView alloc] init];
    self.localVideoView.backgroundColor = [UIColor blackColor];
    
    self.remoteVideoView = [[UIView alloc] init];
    self.remoteVideoView.backgroundColor = [UIColor blackColor];
    
    // Add to view hierarchy
    [self.view addSubview:self.localVideoView];
    [self.view addSubview:self.remoteVideoView];
    
    // Set views in client
    [self.client setLocalViewWithView:self.localVideoView];
    [self.client setRemoteViewWithView:self.remoteVideoView];
}

@end
```

</CodeGroup>

## Conference Room

Join a multi-party conference room:

<CodeGroup>

```swift Swift
let roomId = "conference-room-1"
client.joinRoom(roomId: roomId)

// Handle room events in delegate
func roomJoined(roomId: String) {
    print("Joined room: \(roomId)")
}

func streamJoinedRoom(streamId: String) {
    print("Stream joined: \(streamId)")
    client.play(streamId: streamId)
}

// Leave room
client.leaveFromRoom(roomId: roomId)
```

```objectivec Objective-C
NSString *roomId = @"conference-room-1";
[self.client joinRoomWithRoomId:roomId];

// Handle room events in delegate
- (void)roomJoined:(NSString *)roomId {
    NSLog(@"Joined room: %@", roomId);
}

- (void)streamJoinedRoom:(NSString *)streamId {
    NSLog(@"Stream joined: %@", streamId);
    [self.client playWithStreamId:streamId];
}

// Leave room
[self.client leaveFromRoomWithRoomId:roomId];
```

</CodeGroup>

</CodeGroup>

## Data Channels

Send and receive real-time data:

<CodeGroup>

```swift Swift
// Send data
let message = "Hello viewers!"
client.sendData(message, streamId: streamId)

// Receive data in delegate
func dataReceivedFromDataChannel(_ data: Data, streamId: String) {
    if let message = String(data: data, encoding: .utf8) {
        print("Received: \(message)")
    }
}
```

```objectivec Objective-C
// Send data
NSString *message = @"Hello viewers!";
[self.client sendData:message streamId:streamId];

// Receive data in delegate
- (void)dataReceivedFromDataChannel:(NSData *)data streamId:(NSString *)streamId {
    NSString *message = [[NSString alloc] initWithData:data encoding:NSUTF8StringEncoding];
    NSLog(@"Received: %@", message);
}
```

</CodeGroup>

## Delegate Methods

Implement all delegate methods:

```swift
    func publishFinished(streamId: String)

    func dataReceivedFromDataChannel(_ data: Data, streamId: String)
    func roomJoined(roomId: String)
    func streamJoinedRoom(streamId: String)

}
```

## Requirements

- iOS 12.0 or later
- Xcode 13.0 or later
- Swift 5.5+ or Objective-C
- Camera and microphone hardware
- Ant Media Server with SSL/TLS configured

<Note>
  WebRTC requires HTTPS in production. Make sure your Ant Media Server has a valid SSL certificate.
</Note>
